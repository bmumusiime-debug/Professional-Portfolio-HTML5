# Professional Semantic Portfolio

This repository contains a 5-page HTML5 portfolio created for the Web Design and Development course unit.

## Pages

- `index.html`
- `about.html`
- `gallery.html`
- `data.html`
- `contact.html`

## Included folders

- `images/`
- `videos/`
- `design/`

## GitHub submission steps

1. Initialize Git in the project folder.
2. Add the files.
3. Commit the work with a clear message.
4. Create a public GitHub repository named `Professional-Portfolio-HTML5`.
5. Push the local project to GitHub.
6. Submit the repository link.
