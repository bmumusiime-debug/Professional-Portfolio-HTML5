Technical Terminology

Semantic HTML
HTML elements that describe the meaning of content, such as header, nav, main, article, and footer.
Accessibility
Designing content so that many different users, including people using assistive technology, can understand and use it.
Relative Hyperlink
A link that connects pages within the same website using file names or folder paths.
Alt Text
Short descriptive text added to images so screen readers and users with slow connections can understand the image purpose.
Wireframe
A simple sketch showing where content and semantic sections will appear on a web page.
